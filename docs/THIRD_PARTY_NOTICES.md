# Third-Party Notices

This file is a working inventory for release preparation. It is not yet a final
legal notice bundle.

## Android / Kotlin

- Android Gradle Plugin (`com.android.application`)
- Kotlin Android Gradle Plugin (`org.jetbrains.kotlin.android`)
- AndroidX Core, Activity Compose, Lifecycle, Compose Foundation/UI/Material3
- Kotlin Coroutines
- JUnit
- org.json JSON library for JVM unit tests

## Maps / Navigation

- MapLibre GL Native Android SDK (`org.maplibre.gl:android-sdk`)
- osmdroid Android (`org.osmdroid:osmdroid-android`)
- Free Nautical Chart / QMAP DE online raster tiles are used as a free chart
  source for German waters. Product copy must keep the upstream disclaimer:
  informational/reference/training use only, not navigation.
- OpenStreetMap standard tiles are used only as an online orientation fallback
  under OpenStreetMap attribution and tile-usage policy. Do not bulk-prefetch
  or market them as a sea-chart package.
- OpenSeaMap seamark tiles are used as an online seamark overlay.
- Natural Earth `world_land_110m.geojson` style/source material must retain its
  upstream attribution before store release.

## Billing

- Google Play Billing Library (`com.android.billingclient:billing`)

## Fonts / UI Assets

- Bundled fonts and generated seaFOX UI assets require a final license and
  provenance check before public distribution or store release.

## Marine Chart Data

- NOAA/OpenSeaMap/QMAP DE/ENC/S-57 style sources and any future S-63/C-Map providers
  require provider-specific attribution, license checks, and entitlement
  separation before commercial use.
