legal notice bundle.

- Android Gradle Plugin (`com.android.application`)
- Kotlin Android Gradle Plugin (`org.jetbrains.kotlin.android`)
- AndroidX Core, Activity Compose, Lifecycle, Compose Foundation/UI/Material3
- Kotlin Coroutines
- JUnit

## Maps / Navigation

- MapLibre GL Native Android SDK (`org.maplibre.gl:android-sdk`)
- osmdroid Android (`org.osmdroid:osmdroid-android`)
- Natural Earth `world_land_110m.geojson` style/source material must retain its
  upstream attribution before store release.

- Google Play Billing Library (`com.android.billingclient:billing`)

## Fonts / UI Assets

## Marine Chart Data

- NOAA/OpenSeaMap/ENC/S-57 style sources and any future S-63/C-Map providers
  require provider-specific attribution, license checks, and entitlement
  separation before commercial use.
