# Raw Sources

Hier liegen unveraenderte externe Quellen fuer das LLM-Wiki: Web-Clips, PDF-Exporte, Meetingnotizen, Screenshots, Datenblaetter oder Recherche-Artefakte.

Regeln:

- Nach dem Ablegen nicht nachtraeglich bearbeiten.
- Updates als neue Datei mit Datum speichern, z.B. `2026-04-24-provider-notes.md`.
- Grosse Binaerdateien nur aufnehmen, wenn sie fuer das Projekt wirklich gebraucht werden.
- Zu jeder verarbeiteten Quelle eine Zusammenfassung unter `../wiki/sources/` anlegen.

Interne Projektdateien bleiben an ihrem normalen Ort im Repo und werden direkt referenziert.
